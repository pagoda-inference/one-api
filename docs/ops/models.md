# 模型管理

---

## 模型列表

进入 **运营管理 → 模型管理** 查看所有模型。

### 列表字段

| 字段 | 说明 |
|:-----|:-----|
| 模型 ID | 唯一标识，API 调用使用 |
| 名称 | 显示名称 |
| Provider | 所属供应商 |
| 类型 | 模型分类 |
| 上下文 | 最大输入 Token |
| 输入价格 | 元 / 千 Token |
| 输出价格 | 元 / 千 Token |
| RPM | 每分钟请求上限 |
| 状态 | 启用/禁用 |

---

## 添加模型

1. 进入 **模型管理** 页面
2. 点击 **添加模型**
3. 填写配置：

| 配置项 | 说明 | 必填 |
|:------|:-----|:----:|
| 模型 ID | 与上游一致的标识 | ✓ |
| 名称 | 显示名称 | ✓ |
| Provider | 选择所属供应商 | ✓ |
| 类型 | 对话/视觉/Embedding 等 | ✓ |
| 上下文长度 | 最大输入 Token 数 | ✓ |
| 输入价格 | 元 / 千 Token | ✓ |
| 输出价格 | 元 / 千 Token | ✓ |

---

## 计费公式

```
消耗额度 = (输入 Token × 输入价格 / 1000 + 输出 Token × 输出价格 / 1000) × 分组倍率
```

**示例：**

| 项目 | 值 |
|:----|:---|
| 输入 Token | 1000 |
| 输出 Token | 500 |
| 输入价格 | 0.1 元/K |
| 输出价格 | 0.2 元/K |
| 分组倍率 | 1.0 |

```
消耗 = (1000 × 0.1 / 1000 + 500 × 0.2 / 1000) × 1.0 = 0.2 元
```

## 限流配置

| 参数 | 说明 | 单位 |
|:-----|:-----|:-----|
| RPM | 每分钟请求数上限 | 次/分钟 |
| TPM | 每分钟 Token 上限 | Token/分钟 |

## 批量操作

支持批量启用、禁用、删除模型：

1. 勾选目标模型
2. 选择批量操作
3. 确认执行
