## 快速配置

### 1. 配置渠道

在 **运营管理 → 渠道管理** 中添加上游渠道：

1. 选择渠道类型（OpenAI兼容/Anthropic/自定义）
2. 填写 Base URL 和 API Key
3. 配置模型映射
4. 设置优先级

### 2. 配置模型

在 **模型管理** 中添加模型：

1. 填写模型 ID 和名称
2. 选择 Provider
3. 设置输入/输出价格
4. 配置限流参数（RPM/TPM）

### 3. 配置配额

在 **运营管理 → 配额设置** 中设置：

1. 新用户初始配额
2. 邀请奖励配额
3. 预消耗配额

### 4. 系统设置

在 **运营管理 → 系统设置** 中调整：

1. 分组倍率 - 不同用户分组的价格倍率
2. 渠道自动管理 - 自动禁用/启用渠道
3. 告警设置 - 邮件和 Webhook 通知
4. 性能配置 - 并发、队列、超时参数
