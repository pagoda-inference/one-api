# Provider 管理

Provider 是模型的上游服务实体，管理和维护供应商信息。

---

## Provider 列表

进入 **运营管理 → Provider 管理** 查看所有供应商。

### 列表字段

| 字段 | 说明 |
|:-----|:-----|
| 名称 | Provider 名称 |
| Logo | 供应商 Logo |
| 渠道数 | 关联的渠道数量 |
| 模型数 | 关联的模型数量 |
| 状态 | 启用/禁用 |

---

## 添加 Provider

1. 进入 **Provider 管理** 页面
2. 点击 **添加 Provider**
3. 填写配置：

| 配置项 | 说明 | 必填 |
|:------|:-----|:----:|
| 名称 | Provider 名称 | ✓ |
| Logo | 上传 Logo 图片 | - |
| 描述 | 简介说明 | - |

---

## 关联渠道和模型

添加 Provider 后，需要：

1. **创建渠道时** - 选择关联的 Provider
2. **创建模型时** - 选择所属 Provider

> [!TIP]
> 关联后便于按供应商筛选和管理模型。

---

## 状态管理

| 状态 | 说明 |
|:-----|:-----|
| 启用 | 正常可用 |
| 禁用 | 暂停使用，下属渠道和模型都不可用 |

> [!WARNING]
> 禁用 Provider 后，其下所有渠道和模型都将暂停服务。
