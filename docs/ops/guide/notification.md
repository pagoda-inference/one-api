# 系统通知

管理员可以向用户发送系统通知。

---

## 发送通知

### 发送流程

1. 进入 **运营管理 → 系统通知**
2. 点击 **发送通知**
3. 填写内容：

| 字段 | 说明 | 必填 |
|:-----|:-----|:----:|
| 标题 | 通知标题 | ✓ |
| 内容 | 通知正文 | ✓ |
| 范围 | 全部用户/指定用户 | ✓ |

4. 点击 **发送**

---

## 通知类型

| 类型 | 说明 | 适用场景 |
|:-----|:-----|:---------|
| 系统公告 | 重要系统通知 | 维护通知、更新公告 |
| 活动通知 | 运营活动 | 优惠活动、礼品赠送 |
| 维护通知 | 系统维护 | 计划停机通知 |

---

## 常用模板

### 维护通知

```
标题：系统维护通知
内容：
各位用户，系统将于 XX 月 XX 日 XX:00 进行维护，预计持续 X 小时。
维护期间服务暂停使用，给您带来不便敬请谅解。
```

### 活动通知

```
标题：限时优惠活动
内容：
即日起至 XX 月 XX 日，充值满 100 元赠送 20 元！
快去充值中心参与活动吧。
```

> [!TIP]
> 定期发送系统通知可以提高用户活跃度。
