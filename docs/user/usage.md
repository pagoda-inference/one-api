# 用量明细

查看 API 使用情况和消耗统计。

---

## 用量汇总

在 **用量明细** 页面顶部查看汇总数据：

| 指标 | 说明 |
|:-----|:-----|
| 总请求数 | 累计 API 调用次数 |
| 总消耗额度 | 累计消耗的额度（Quota） |
| 总输入 Token 数 | 累计输入内容消耗的 Token 总数 |
| 总输出 Token 数 | 累计模型返回内容消耗的 Token 总数 |

> [!TIP]
> 额度不足时及时充值，避免影响业务。

---

## 按模型统计

点击 **按模型** 标签查看各模型使用情况：

| 字段 | 说明 |
|:-----|:-----|
| 模型名称 | 使用的模型 |
| 请求数 | 调用次数 |
| 消耗额度 | 消耗的 Quota |
| 输入 Token | 累计输入 Token |
| 输出 Token | 累计输出 Token |

---

## 用量趋势

查看近期的用量趋势图表：

- **按日统计** - 每日用量变化
- **按模型统计** - 各模型用量占比

---

## 日期筛选与模型筛选

页面顶部提供**日期选择器**与**模型选择器**，支持单独筛选或组合筛选统计数据：
- **日期选择器**：自定义选择起止日期，灵活锁定统计时间范围
- **模型选择器**：可单独选择指定模型，查看单模型用量统计
- **组合筛选**：支持同时选择日期范围与目标模型，精准获取指定模型在对应时间区间内的用量数据

选择完成后，点击「查询」按钮即可刷新对应统计数据。

---

## 每日明细

使用 **日期筛选与模型筛选** 后查看详细调用记录：

| 字段 | 说明 |
|:-----|:-----|
| 时间 | 请求时间 |
| 模型 | 使用的模型 |
| 请求数 | 调用次数 |
| 消耗额度 | 实际消耗的 Quota |
| 输入 Token | 消耗的输入 Token |
| 输出 Token | 消耗的输出 Token |

> [!NOTE]
> 明细记录保留最近 30 天的数据。

同时可以查看**每日用量趋势图**，以柱状图形式直观展示每日输入Token、输出Token的使用变化，可清晰查看用量波动情况，支持按日期追溯每日Token消耗明细。

## 常见问题

**Q: 为什么显示的用量和实际不符？**

A: 可能原因：
- 预扣额度与实际消耗有差异（已调整）
- 部分请求失败未计入
- 并非所有模型都启用用量记录

**Q: 如何降低用量？**

A: 建议：
- 适当降低 `max_tokens` 限制
- 减少不必要的上下文对话
- 使用流式输出（streaming）
