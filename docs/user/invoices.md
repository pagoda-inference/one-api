# 发票管理

---

## 申请发票

### 申请条件

> [!NOTE]
> 累计充值满一定金额后可申请发票，具体金额限制请咨询客服。

### 发票类型

| 类型 | 说明 | 审核时间 |
|:-----|:-----|:---------|
| 增值税普通发票 | 开票快，一般 1-2 个工作日 | 1-2 天 |
| 增值税专用发票 | 需要资质审核，一般 3-5 个工作日 | 3-5 天 |

---

## 申请流程

1. 进入 **发票管理** 页面
2. 点击 **申请发票**
3. 填写发票信息：
   - 发票类型（普票/专票）
   - 发票抬头
   - 纳税人识别号
   - 开票金额
   - 接收邮箱
4. 提交申请

### 普票所需信息

| 字段 | 说明 |
|:-----|:-----|
| 发票抬头 | 公司或个人名称 |
| 纳税人识别号 | 税务登记号 |

### 专票所需信息

| 字段 | 说明 |
|:-----|:-----|
| 发票抬头 | 公司名称 |
| 纳税人识别号 | 税务登记号 |
| 开户银行 | 对公账户开户行 |
| 银行账号 | 对公账户号 |
| 注册地址 | 公司注册地址 |
| 注册电话 | 公司联系电话 |

---

## 发票状态

| 状态 | 说明 | 可执行操作 |
|:-----|:-----|:-----------|
| 申请中 | 等待审核 | 取消 |
| 开票中 | 正在开具发票 | 查看进度 |
| 已开票 | 开票完成，发送至邮箱 | 下载 |
| 已作废 | 发票已作废 | 重新申请 |

---

## 常见问题

**Q: 发票多久能收到？**

A: 普通发票 1-2 个工作日，专用发票 3-5 个工作日。

**Q: 发票可以冲红吗？**

A: 可以，请联系客服处理。
