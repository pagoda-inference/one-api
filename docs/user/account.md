# 账户管理

---

## 登录方式

系统支持多种登录方式：

| 登录方式 | 说明 | 状态 |
|:---------|:-----|:-----|
| 飞书登录 | 企业飞书账号 | 默认开启 |
| 密码登录 | 邮箱 + 密码 | 默认开启 |
| GitHub 登录 | GitHub 账号 | 请联系管理员 |
| OIDC 登录 | 企业 OIDC 单点登录 | 请联系管理员 |

> [!NOTE]
> 具体开启哪些登录方式，由系统配置决定。如需开通请咨询运维人员。

---

## 个人信息

进入 **个人中心** 页面管理个人信息：

| 可修改项 | 说明 |
|:--------|:-----|
| 显示名称 | 界面中显示的名称 |
| 邮箱 | 账号绑定邮箱（修改后需重新验证） |
| 密码 | 登录密码 |

---

## 角色权限

| 角色 | 权限等级 | 说明 |
|:-----|:--------|:-----|
| 普通用户 | 0 | 使用 API、充值、管理自己的 Key |
| 管理员（暂未开放） | 10 | 管理渠道、模型、系统设置 |
| Root | 100 | 超级管理员，拥有所有权限 |

> [!TIP]
> 普通用户如需管理员权限，请联系 Root 用户升级。

---

## 修改密码

1. 进入 **个人中心**
2. 点击 **修改密码**
3. 填写表单：
   - 当前密码
   - 新密码（8位以上）
   - 确认新密码
4. 点击 **确认修改**

> [!WARNING]
> 修改密码后需要重新登录，请牢记新密码。

---

## 安全设置

> [!DANGER]
> 发现账号异常请立即联系管理员，或修改密码。

- 定期更换密码
- 不要在多个网站使用相同密码
- 不要将密码告诉他人
