# 签到奖励

每日签到可获得额度奖励，轻松获取免费使用时长。

---

## 签到规则

| 规则 | 说明 |
|:-----|:-----|
| 签到次数 | 每日仅可签到一次 |
| 断签重置 | 连续签到中断后重新计算 |
| 奖励发放 | 签到成功后即时到账 |

---

## 连续签到奖励

连续签到天数越多，奖励越丰厚：

| 连续天数 | 奖励倍率 | 说明 |
|:--------:|:--------:|:-----|
| 1-7 天 | 1x | 基础奖励 |
| 8-14 天 | 1.5x | 中级奖励 |
| 15-30 天 | 2x | 高级奖励 |
| 31 天以上 | 3x | 特级奖励 |

> [!TIP]
> 坚持签到，奖励翻倍！

---

## 签到流程

1. 进入首页或签到页面
2. 点击 **签到** 按钮
3. 签到成功，获得相应奖励
4. 查看连续签到天数和累计奖励

---

## 查看记录

在签到页面查看：

| 记录项 | 说明 |
|:-------|:-----|
| 今日签到状态 | 已签到/未签到 |
| 连续签到天数 | 当前连续签到天数 |
| 累计获得奖励 | 历史签到获得的总奖励 |

---

## 常见问题

**Q: 可以补签吗？**

A: 不支持补签，请每日按时签到。

**Q: 签到奖励会过期吗？**

A: 签到奖励获得的额度无有效期限制，可正常使用。
