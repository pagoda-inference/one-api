## 快速开始

1. **注册账号** - 使用邮箱或 OAuth 方式注册
2. **获取 API Key** - 在 API Keys 页面创建密钥
3. **充值额度** - 在充值中心购买额度
4. **开始使用** - 调用 API 使用 AI 能力
