# 充值中心

---

## 充值方式

支持多种充值方式：

| 支付方式 | 说明 | 到账速度 |
|:---------|:-----|:---------|
| 支付宝 | 扫码支付 | 即时到账 |
| 微信支付 | 扫码支付 | 即时到账 |

> [!NOTE]
> 如需其他支付方式，请联系客服。

---

## 充值金额

### 快速充值

预设金额选项，方便快捷：

| 金额 | 赠送 | 实际到账 |
|:----:|:----:|:--------:|
| ¥10 | - | ¥10 |
| ¥50 | - | ¥50 |
| ¥100 | - | ¥100 |
| ¥200 | - | ¥200 |
| ¥500 | - | ¥500 |

### 自定义金额

选择 **自定义金额**，输入想充值的金额。

> [!WARNING]
> 最小充值金额为 ¥1。

---

## 充值流程

1. 选择充值金额或输入自定义金额
2. 选择支付方式（支付宝/微信）
3. 点击 **立即充值**
4. 扫描二维码完成支付
5. 支付成功后，额度**自动到账**

> [!DANGER]
> 请在 **30 分钟内** 完成支付，超时后订单自动取消。

---

## 订单管理

在 **充值中心** 页面查看充值记录：

| 状态 | 说明 | 操作 |
|:-----|:-----|:-----|
| 待支付 | 订单已创建，等待支付 | 继续支付 |
| 已支付 | 支付成功，额度已到账 | 查看 |
| 已取消 | 订单超时或用户取消 | 重新充值 |

### 订单信息

| 字段 | 说明 |
|:-----|:-----|
| 订单号 | 唯一订单标识 |
| 金额 | 充值金额 |
| 支付方式 | 支付宝/微信 |
| 创建时间 | 订单创建时间 |
| 支付时间 | 完成支付时间 |

## 额度说明

> [!TIP]
> 充值金额 = 账户额度，1 元 = 7200 Quota。

**额度用途：**
- 支付 API 调用费用
- 按实际消耗扣除

## 常见问题

**Q: 支付成功后额度没到账怎么办？**

A: 请稍等几分钟，如仍未到账请联系客服并提供订单号。

**Q: 可以开发票吗？**

A: 可以，请在 **发票管理** 页面申请。
